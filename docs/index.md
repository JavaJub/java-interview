## Подготовка по времени

=== "3 дня"

    Выберите 1–2 релевантных гайда, пройдите чек-листы, повторите HashMap, `equals/hashCode`, Stream API, Spring DI, SQL JOIN/индексы. Прогоните Express-тест на 15 вопросов.

=== "7 дней"

    Добавьте live-coding, 20–30 задач Easy/Medium, повторите транзакции, Kafka, JMM, Docker/Kubernetes по стеку вакансии. Standard-тест на 30 вопросов по слабым темам.

=== "14 дней"

    Пройдите маршрут по грейду, соберите ответы на поведенческие вопросы, потренируйте mock interview и разбор кода вслух. Interview-simulation на 60 вопросов.

## Как пользоваться

Откройте гайд по нужной компании и идите сверху вниз. Подсказки под вопросами — это не учебник, а быстрый способ вспомнить ответ по ключевым словам. Если собеседование уже близко, сначала пройдите финальный чек-лист и задачи, а затем закрепите слабые темы в [тестах JavaJub](quizzes/index.md).

Это русскоязычная база для подготовки к Java собеседованию, которая закрывает типовые запросы: вопросы Java-собеседования, подготовка Java Middle, Spring Boot вопросы, Kafka и SQL на интервью, live coding на Java.

## Java вакансии и подготовка под Job

Если вы просматриваете Java вакансии, начните с [разбора требований вакансии](jobs.md): там показано, как переводить стек из описания вакансии в конкретный план подготовки. Например, `Spring Boot + PostgreSQL + Kafka` превращается в повторение DI, транзакций, индексов, consumer groups и delivery guarantees, а не в абстрактное «повторить Java».
